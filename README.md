# Voice-AI

[Live Demo](https://voiceapp-ai.vercel.app/)

## Overview

Voice-AI is an AI-powered application combining a modern frontend with a robust backend system. Its goal is to deliver advanced voice interaction capabilities, allowing users to communicate effectively with the application using speech.

## Features

- Speech recognition and processing
- AI-powered responses or actions based on voice input
- Modular backend and frontend architecture
- Powered by TypeScript for type-safe development

## Project Structure

- `/frontend`: The UI and user experience, built for smooth voice interactions.
- `/backend`: APIs and logic for processing voice commands and handling AI operations.

## Getting Started

### Prerequisites

- Node.js (v16 or above)
- npm

### Installation

```bash
git clone https://github.com/sujitkumar-13/Voice-AI.git
cd Voice-AI
# Install frontend dependencies
cd frontend
npm install
# Install backend dependencies
cd ../backend
npm install
```

### Running the Application

Start the backend and frontend separately:

```bash
# Backend
cd backend
npm start

# Frontend
cd ../frontend
npm run dev
```

## Usage

1. Open the app via [Live Demo](https://voiceapp-ai.vercel.app/) or run locally.
2. Interact using your voice; the AI will process and respond accordingly.
